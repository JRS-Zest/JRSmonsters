ステータス（STR / AGI / CON / WIS / INT / CHA / LCK）
stat = floor( 基礎値 × SF/100000 × (Lv−1) + 基礎値 )
SF = StatusFactor（種族固定値）
MaxHP
MaxHP = floor( lb/10 × Lv + dhp/100 + sb/10 × CON )
lb = LevelUpBonus　dhp = DefaultHP　sb = StateBonus
MaxCP
MaxCP = floor( lb/10 × Lv + dcp/100 + sb/10 × CHA )
dcp = DefaultCP（HP と lb/sb を共有）
攻撃力
Atk = ( AtkBonusScale/100 × (Lv−1) + AtkBase ) × ( 1 + STR/200 )
上限: bhi/AtkHi　下限: blo/AtkLo　で個別計算
防御力
Def = ( db/100 × (Lv−1) + DefBase ) × ( 1 + CON/100 )
db = DefenceValueBonus　DefBase = DefenceValue
魔法抵抗（火/水/風/土/光/闇）
MagRes[i] = 基礎値[i] + floor( WIS / 30 )
WISボーナスは全属性に一律加算（レベル依存あり）
状態異常抵抗
AbnRes[i] = 固定値（レベル・ステータス依存なし）
全状態異常共通耐性・全呪文耐性・全衰退耐性も同様
